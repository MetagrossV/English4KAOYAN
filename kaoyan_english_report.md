
# 考研英语真题爬取报告

## 一、网站探测结果

### 1. 基本信息
- **网站URL**: https://pastpapers.cn/kaoyan
- **网站名称**: 在线真题 - 历年考研英语真题
- **技术架构**: Next.js (React SSR/CSR)
- **内容加载**: 客户端JavaScript动态加载 + 滚动加载

### 2. 年份范围
- **最早年份**: 1998年
- **最新年份**: 2025年
- **总计年份**: 1998-2025 (28年)

### 3. 试卷结构
- **2010-2025年**: 每年2套试卷
  - `-1`: 英语一 (考研英语一)
  - `-2`: 英语二 (考研英语二)
- **2009年及以前**: 每年1套试卷（不分英语一/二）

### 4. 链接格式
- **相对路径**: `/paper/{year}-1` 或 `/paper/{year}-2`
- **完整URL**: `https://pastpapers.cn/paper/{year}-1`

### 5. 页面特征
- 使用**无限滚动**加载更多年份
- PDF文件需要**登录**才能下载
- 试卷内容通过**客户端渲染**显示
- 每个试卷有独立的PDF文件，URL格式: `/uploads/{timestamp}_{random}.pdf`

## 二、API端点

通过Playwright网络拦截发现的API:
- `GET /api/translation-config` - 翻译配置
- `GET /api/config?t={timestamp}` - 网站配置
- `GET /api/auth/domains/public?t={timestamp}` - 认证域
- `GET /api/user/me?t={timestamp}` - 用户信息

## 四、数据结构说明

JSON文件包含以下字段:
- `paper_id`: 试卷ID (如: 2025-1, 2024-2)
- `year`: 年份
- `exam_type`: 考试类型 (考研英语一/考研英语二)
- `source_url`: 来源页面URL
- `pdf_url`: PDF文件URL
- `translation`: 翻译题目
  - `type`: 翻译类型 (英译汉/英译汉(全文翻译))
  - `word_count_approx`:  approximate词数
  - `instructions`: 翻译指令
  - `original_text`: 英文原文
- `writing`: 写作题目
  - `part_a`: 小作文
    - `type`: 应用文类型 (书信/邮件/通知等)
    - `word_count`: 字数要求
    - `instructions`: 写作指令
    - `topic`: 主题
  - `part_b`: 大作文
    - `type`: 作文类型 (图表作文/图画作文)
    - `word_count`: 字数要求
    - `instructions`: 写作指令
    - `image_description`: 图表/图画描述

## 五、技术说明

### 爬取方法
1. 使用**Playwright**模拟浏览器获取PDF URL
2. 使用**Requests**下载PDF文件
3. 使用**PyMuPDF (fitz)**解析PDF文本
4. 使用**正则表达式**提取翻译和写作内容

### 遇到的挑战
1. **动态加载**: 页面内容通过JavaScript动态加载，需要使用浏览器自动化工具
2. **格式差异**: 英语一和英语二的试卷格式不同
   - 英语一: Part C划线翻译, 51/52题号
   - 英语二: 全文翻译, 47/48题号
3. **PDF提取**: 部分PDF的文本提取不完整，存在换行和格式问题

### 改进方向
1. 使用更精确的PDF解析方法，保留原文格式
2. 针对英语二的格式优化提取逻辑
3. 添加图表数据的结构化提取
4. 处理2009年及以前的试卷格式
